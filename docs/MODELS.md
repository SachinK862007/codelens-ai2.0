## Codelens Models

### Model 1: Codelens Code Runner
- Accepts code in Python/C/C++ and executes it locally.
- Returns stdout/stderr, a placeholder algorithm flowchart, and playback steps.
- If errors occur, returns a simple suggested code template.

### Model 2: Codelens Debugger
- Accepts code or file upload.
- Runs code locally and explains syntax/runtime/compiler errors.
- Provides step-by-step guidance and a basic fix template.

### Model 3: Codelens Idea Builder
- Accepts a project idea description.
- Returns a structured blueprint: problem, resources, stack, and file tree.

### Model 4: Codelens Learner
- Five levels of beginner challenges.
- Validates by running code and checking expected output.
- Unlocks the next level on success.
